Get the zip build for x64 Linux.

Download .zip

Tested on Ubuntu 20.04+.

Quick start
  1. Download the zip, extract the zip.
  2. Make "launchgame.sh" executable.
  3. Run "launchgame.sh" from your file manager.
zip build

You can generate a new build with `scripts/build-zip.sh`.

Need help?

If the file won't run, check permissions or use a different distro build.
