Protocol registration
=====================

These scripts register a custom URL protocol to launch games via launchgame.sh.
Protocol format: benno111engene://<slug>/<version>

Linux:
  ./client/protocol/register-linux.sh benno111engene

macOS:
  ./client/protocol/register-macos.sh benno111engene

Windows:
  client\\protocol\\register-windows.bat benno111engene
